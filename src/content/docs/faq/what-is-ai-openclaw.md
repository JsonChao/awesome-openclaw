# 什么是 ai openclaw

ai openclaw 是一个可扩展的 agent 系统，不是单纯聊天机器人。它的核心是“理解任务 + 调用能力 + 执行动作 + 返回结果”。

## 三个差异点

1. 可执行：ai openclaw 能连接工具并执行任务。
2. 可扩展：ai openclaw 可安装技能与自定义流程。
3. 可运营：ai openclaw 可监控、可审计、可迭代。

## 适用场景

- 个人效率增强。
- 团队信息协同。
- 业务流程自动化。
