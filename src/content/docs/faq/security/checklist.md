# ai openclaw 安全检查清单

## 上线前

1. 所有密钥已迁移到安全存储。
2. 默认凭证与默认端口已修改。
3. 高危技能已禁用或隔离。
4. 日志与告警渠道可用。

## 每周巡检

1. 检查异常调用与失败任务。
2. 检查新增技能权限变更。
3. 检查依赖与镜像更新。

## 每月演练

1. 恢复演练。
2. 漏洞响应演练。
3. 访问控制抽查。
