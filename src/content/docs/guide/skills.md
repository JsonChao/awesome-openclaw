# ai openclaw 技能市场与自定义技能

技能可以快速扩展 ai openclaw，但也可能成为风险入口。

## 安装前检查

1. 来源是否可信。
2. 权限是否最小化。
3. 是否读取敏感目录或变量。
4. 是否有活跃维护与更新记录。

## 自定义技能建议

- 输入输出结构化。
- 错误可观测。
- 权限可审计。
- 版本可回滚。

强烈建议搭配 [安全总览](/guide/security) 一起实施。
