# 内容团队自动化模板

这是一个可直接复用的 ai openclaw 模板：

1. 每天抓取行业信息并聚合。
2. 自动生成选题建议与优先级。
3. 生成初稿大纲并通知编辑复核。
4. 发布后自动汇总表现数据。

## 价值

- 降低重复劳动。
- 提升内容节奏稳定性。
- 让编辑专注在判断与质量。
